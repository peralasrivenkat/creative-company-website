# A Creative Company Website

## Project Description
This project is a professional and responsive website for a creative company.  
It is built using **HTML, CSS, and JavaScript** and features interactive elements to enhance user experience.  
The website is fully responsive and works across desktops, tablets, and mobile devices.

## Features
- Fully responsive design using CSS media queries
- Interactive elements such as buttons, sliders, and navigation menus
- Smooth animations and transitions
- Clean and modern layout suitable for a creative company
- Optimized for usability and accessibility

## Tools & Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- Netlify for deployment

## Deployment
The website is deployed on Netlify: [Live Demo](https://lustrous-gecko-356c9d.netlify.app/)
